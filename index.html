01_Project: Efficiency Improvement

INPUT / ISSUE

手待ち時間の発生
(非効率なアイドルタイム)

PROCESS / ACTION

セルフタスクリスト
(ルーティン化・アルゴリズム構築)

OUTPUT / RESULT

退勤時間の短縮
(システム全体の最適化)

飲食店のホールスタッフとして、業務プロセスの最適化に取り組みました。 私の店舗では、注文が途切れた際にスタッフが稼働しない「手待ち時間（アイドルタイム）」が発生しており、これが全体のスループット低下の要因（課題）であると分析しました。


そこで私は、「単独でもホール運営を完結可能な状態にする」ことを目標に設定し、アイドルタイムを活用する「セルフタスクリスト」を実装・実行しました。 具体的には、「洗い場のスタック確認」「リソース（備品）の補充」「終了処理（閉店準備）の前倒し」という一連のタスクをアルゴリズムのようにルーティン化し、徹底しました。


このプロトコルを継続した結果、他のスタッフが管理業務（レジ締め）を行っている並行処理の間に、私一人でもホールのクローズ処理を完了できるようになり、店舗全体の稼働時間短縮（コスト削減）に貢献できました。 この経験から、現状のシステムにおけるボトルネックを発見する分析力と、自ら改善策を実装する主体性を習得しました。
